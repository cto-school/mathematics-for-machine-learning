# Mathematics for Machine Learning — A Hands-On Course

A from-scratch course that takes a **mathematics student with little or no
programming experience** all the way from "what is a variable?" to
**building a neural network by hand**.

The philosophy: you already know (or are learning) the math. We connect every
equation to a few lines of Python so the math becomes something you can *run,
plot, and poke at*. Code is kept deliberately simple; explanations are short
unless a topic genuinely needs depth.

---

## Who this is for

- You are comfortable with high-school / early-university math (algebra,
  functions, a little calculus). Don't worry if calculus and linear algebra are
  rusty or new — we (re)build the intuition here.
- You have **zero or basic** Python / computer experience. Chapter 0 assumes
  nothing.

## What you will be able to do at the end

- Use **NumPy** fluently to express vectors, matrices, and equations.
- **Visualize** functions, data, and models with Matplotlib.
- Read and use the core math of ML: **linear algebra, calculus, gradients,
  probability**.
- Implement, *from scratch*, **linear regression, logistic regression,
  gradient descent, and a small neural network** — and understand every line.

---

## How each chapter is organized

Each chapter folder contains:

- **`README.md`** — the necessary theory, in compact form (the "why").
- **One or more `.ipynb` notebooks** — heavily commented, runnable code (the
  "how"), with:
  - worked **Exercises** that include their **solutions** in the notebook, and
  - **Homework** problems at the end whose solutions are *not* given — try them!

> Notebooks are generated from the small scripts in `tools/generators/` using
> the helper `tools/nbutil.py`. You normally just open the `.ipynb` files; the
> generators are there so the whole course is reproducible.

## Setup (one time)

```bash
pip install -r requirements.txt
jupyter notebook        # or:  jupyter lab
```

Then open any `.ipynb` and run cells top to bottom with **Shift+Enter**.

## Curriculum

### Part I — Tools of the Trade
| # | Chapter | You will learn |
|---|---------|----------------|
| 00 | **Python & Jupyter Crash Course** | variables, types, lists, loops, functions, and how notebooks work — framed for mathematicians |
| 01 | **NumPy for Mathematics** | arrays as vectors/matrices, vectorization, broadcasting, math functions, random numbers |
| 02 | **Visualization with Matplotlib** | plotting functions, curves, scatter data, surfaces & contours — turning equations into pictures |

### Part II — The Core Mathematics
| # | Chapter | You will learn |
|---|---------|----------------|
| 03 | **Linear Algebra I — Vectors** | vectors, length/norm, dot product, projections, angles, span |
| 04 | **Linear Algebra II — Matrices & Linear Systems** | matrices as transformations, matrix multiply, inverse, solving `Ax = b`, determinant, rank |
| 05 | **Linear Algebra III — Eigenvalues, SVD & PCA** | eigenvectors, diagonalization, the SVD, and PCA for dimensionality reduction |
| 06 | **Calculus & Derivatives** | limits, derivatives (numeric & symbolic intuition), rules, Taylor approximation, finding minima |
| 07 | **Multivariate Calculus & Gradients** | partial derivatives, the gradient, chain rule, Jacobians — the engine of learning |
| 08 | **Probability & Statistics** | random variables, distributions, mean/variance, Bayes' rule, sampling, the normal distribution |

### Part III — Machine Learning, Built by Hand
| # | Chapter | You will learn |
|---|---------|----------------|
| 09 | **Introduction to Machine Learning** | what learning *is*, features/labels, train vs test, loss functions, over/underfitting |
| 10 | **Linear Regression** | fitting a line, least squares (closed form), the cost surface, evaluation |
| 11 | **Logistic Regression & Classification** | the sigmoid, decision boundaries, cross-entropy, accuracy — **plus full multi-class softmax** (notebook 11b) |
| 12 | **Gradient Descent & Optimization** | the workhorse algorithm: learning rates, convergence, stochastic & mini-batch GD |
| 13 | **Neural Networks from Scratch** | the perceptron, a multilayer network, forward pass, **backpropagation** (with an Andrew-Ng-style computation-graph walkthrough), and an **activation-function deep-dive** (notebook 13c) |

### Part IV — Going Further
| # | Chapter | You will learn |
|---|---------|----------------|
| 14 | **Model Evaluation & Regularization** | confusion matrix, precision/recall/F1, ROC; cross-validation; L2/L1 regularization to cure overfitting |
| 15 | **Introduction to CNNs** | convolution & pooling from scratch, edge filters on an image, the conv→ReLU→pool→softmax pipeline (conceptual bridge to deep learning) |

## A note on style

- We prefer **pure NumPy and from-scratch implementations** over black-box
  libraries, because the point is to *understand* the math. Where it's
  illuminating, we briefly compare against `scikit-learn`.
- Every notebook is self-contained and can be run top-to-bottom.

## Further reading (highly recommended)

This course gives you the *math you can run*. To go deeper — especially on neural
networks and backpropagation — these are the classics. We point to the most
relevant one at the end of each chapter's README.

**Andrew Ng (start here for ML/DL):**
- **Machine Learning Specialization** (Coursera, DeepLearning.AI & Stanford) — the
  gentlest on-ramp; mirrors Chapters 9–13 closely.
- **Deep Learning Specialization** (Coursera, DeepLearning.AI) — *Course 1: Neural
  Networks and Deep Learning* is the perfect companion to Chapter 13 (his
  computation-graph view of backprop is the style we adopt here).
- **CS229: Machine Learning** (Stanford) — the [lecture notes](https://cs229.stanford.edu/)
  are a rigorous, math-first treatment.

**Books (free / online):**
- **Mathematics for Machine Learning** — Deisenroth, Faisal & Ong (free PDF at
  [mml-book.com](https://mml-book.com)) — the linear-algebra/calculus/probability
  backbone of Chapters 3–8.
- **Deep Learning** — Goodfellow, Bengio & Courville (free at
  [deeplearningbook.org](https://www.deeplearningbook.org)) — the reference for
  Chapters 13–15.

**Visual intuition:**
- **3Blue1Brown — *Neural Networks*** ([YouTube series](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi))
  — the best visual explanation of gradient descent and backprop anywhere.
- **3Blue1Brown — *Essence of Linear Algebra*** and ***Essence of Calculus*** —
  pair beautifully with Parts I–II.

**Documentation:** [NumPy](https://numpy.org/doc/stable/),
[Matplotlib](https://matplotlib.org/stable/), [scikit-learn](https://scikit-learn.org/stable/).

Happy learning! Start with **`00-python-jupyter-crash-course/`**.
